### HALLAZGOS

#### 1. 🚨 Vercel — Investigación del incidente Abril 2026 sigue expandiéndose (NUEVO desarrollo)
**Severidad:** Media-Alta
**Contexto:** Rod ya respondió al incidente original (commit `d9ae894` "IncidenteVercel" del 2026-04-20). Pero esta semana se publicó "Vercel Finds More Compromised Accounts" — el alcance del breach es mayor de lo inicialmente reportado.

**Recordatorio del breach:** OAuth supply chain attack vía Context.ai. Variables marcadas como "sensitive" están encriptadas y seguras. Variables NO marcadas como sensitive son texto plano y deben rotarse.

**Variables de bondíaCT en posible riesgo (si no están marcadas como sensitive):**
- `VITE_SUPABASE_ANON_KEY` — proyecto ct.bondiact.io
- `RESEND_API_KEY` — proyecto salthub.mx (controla envío de email)
- `RECAPTCHA_SECRET_KEY` — proyecto salthub.mx

**Acción para Rod (sin código):**
1. Abre https://vercel.com/dashboard → entra a cada proyecto (bondiact, control-tower, salthub).
2. Settings → Environment Variables.
3. Para cada variable arriba: confirma que la columna muestra ícono de candado (sensitive).
4. Si NO está marcada como sensitive: clic en los `...` → "Edit" → activa el toggle "Sensitive" → Save. Luego rota la llave en el proveedor (Supabase/Resend/Google reCAPTCHA) y actualízala en Vercel.

#### 2. ⚠️ Supabase — CVE-2026-31813 Auth Bypass (Apple/Azure OAuth)
**Severidad:** Baja para bondíaCT (informativa)
**Contexto:** Vulnerabilidad crítica en Supabase Auth previa a versión 2.185.0 — permite emitir sesiones para usuarios arbitrarios si los providers Apple o Azure están habilitados.

**Impacto en bondíaCT:** Probablemente NULO. CT usa Supabase para base de datos; LIGIE usa Firebase Auth (Google provider). No identifico uso de Apple ni Azure OAuth.

**Acción para Rod (sin código):**
1. Abre https://supabase.com/dashboard → proyecto Control Tower.
2. Authentication → Providers.
3. Verifica que Apple y Azure estén DESACTIVADOS. Si por error alguno está activo, desactívalo o asegúrate de que la versión de la librería esté en 2.185.0 o superior.
